# Code Refactoring Analysis Dashboard

Production-ready React dashboard for visualizing refactoring impact with responsive design, dark mode, and chart-driven analytics.

## Stack

- React (functional components)
- Tailwind CSS
- Recharts
- Vite

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project structure

```text
src/
  components/      # Reusable UI building blocks
  data/            # Mock JSON data source
  hooks/           # Data access abstraction
  sections/        # Dashboard feature sections
  App.jsx          # App shell and section routing
```
